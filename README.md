# 🛒 E-Commerce Application (Backend Focused)

A scalable and secure e-commerce backend application built using Java and Spring Boot. This project demonstrates RESTful API development, authentication, and deployment using modern tools.

---

## 🚀 Features

- User authentication and authorization using JWT
- Role-based access control (Admin/User)
- Product management (Add, Update, Delete, View)
- Cart and order management
- RESTful API design
- Pagination and sorting
- Global exception handling
- Secure backend using Spring Security

---

## 🛠️ Tech Stack

- **Backend:** Java, Spring Boot
- **Security:** Spring Security, JWT
- **Database:** MySQL
- **ORM:** JPA (Hibernate)
- **Build Tool:** Maven
- **DevOps:** Docker
- **Tools:** Git, GitHub, Postman

---

## 📦 Project Structure

- Controller Layer (Handles API requests)
- Service Layer (Business logic)
- Repository Layer (Database interaction)
- Entity Layer (Data models)
